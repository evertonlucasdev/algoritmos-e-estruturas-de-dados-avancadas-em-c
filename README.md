# Algoritmos e Estruturas Avançadas em C

Implementações de alguns algoritmos e aplicações que estudei durante a graduação, envolvendo estruturas de dados, compressão e análise de sequências.

## O que tem aqui

* Sistema de transportadora (processamento de dados/logística)
* Compressão de dados (RLE e Huffman)
* Sequenciamento genético (análise de DNA)

Todos feitos do zero em C, focando em entender como os algoritmos funcionam na prática.

## Estrutura

```
algoritmos-e-estruturas-avancadas-em-c/
    ├── transportadora.c
    ├── compressao.c
    ├── sequenciamento.c
    └── README.md
```

## Como rodar

Exemplo:

```
gcc compressao.c -o compressao
./compressao
```

(os outros seguem o mesmo padrão)
